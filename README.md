## 🔥 Overview
Production-ready E-commerce backend built using Spring Boot with secure authentication, order management, payment integration, and admin capabilities.

## 🚀 Key Features
- 🔐 JWT Authentication (Login, Signup, OTP Verification)
- 👥 Role-Based Access Control (Admin / User)
- 🛍 Product Management with Image Upload
- 🛒 Cart Management & Active Cart Tracking
- ❤️ Wishlist Functionality
- 📦 Order Processing (COD + Online Payment)
- 💳 Razorpay Payment Integration
- 📧 Email Notifications (SMTP)
- 🎟 Promo Code System
- 🔔 Notification System
- 🏠 Address Management
- ⭐ Product Reviews & Ratings
- 🏷 Tag Management (Admin)
- 📊 Admin Dashboard APIs

## 🧩 API Modules
- Authentication (Login, Signup, OTP, Forgot Password, Reset Password)
- Users & Admin Management
- Products & Images
- Cart & Wishlist
- Orders & Checkout
- Promo Codes & Payments
- Notifications
- Reviews & Ratings
- Dashboard

## 🛠 Tech Stack
- Java 17
- Spring Boot 3
- Spring Security
- JWT Authentication
- MySQL
- Redis
- Docker
- Cloudinary (Image Upload)
- Razorpay (Payment Gateway)
